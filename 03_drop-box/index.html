How to centralize content in different type of block? - text content in block-level elements centralized horizontally by text-align:center. - text content in inline-level elements no need to centralize, the size of inline-level is support by its content. - inline-block inside a block-level element can be treated as text content. - block-level inside a defined size(width and height) of block-level element normally use positioning to centralize it.
